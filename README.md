# MediaHub - Premium Social Media Marketplace

MediaHub is a modern, high-end marketplace for buying premium social media accounts. Designed with a focus on UI/UX, animations, and 3D visual effects.

## Project Structure

```text
MediaHub/
├── index.html          # Homepage with Hero section and 3D visuals
├── marketplace.html    # Product listing with category tabs
├── about.html          # Company information and vision
├── contact.html        # Customer support and contact form
├── css/
│   └── style.css       # Core styling, light effects, and responsiveness
├── js/
│   └── script.js       # Animations (GSAP) and 3D rendering (Three.js)
├── assets/             # Images and icons (folder created)
└── README.md           # Project documentation
```

## Features

- **Modern UI/UX**: Dark theme with neon glassmorphism effects.
- **Background Light Effects**: Dynamic floating orbs for a futuristic feel.
- **3D Animations**: Interactive 3D elements powered by Three.js in the hero section.
- **Smooth Transitions**: GSAP-powered text and element animations.
- **Responsive Design**: Fully functional on mobile, tablet, and desktop.
- **Secure Marketplace Layout**: Pre-designed cards for Instagram, Twitter, TikTok, and YouTube accounts.

## Tech Stack

- **Frontend**: HTML5, CSS3 (Custom Properties, Flexbox, Grid)
- **Animations**: GSAP (GreenSock Animation Platform)
- **3D Graphics**: Three.js (WebGL)
- **Icons**: Font Awesome 6
- **Typography**: Google Fonts (Poppins)

## How to Run

Simply open `index.html` in any modern web browser.
